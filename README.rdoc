## Artnet Auctions Clone

A version of http://www.artnet.com/auctions/ built with Ruby on Rails.

### Setup
clone project onto your desktop and go to the project directory in the terminal.
run postgres
```
$ postgres
```
create the database:
```
$ rake db:create
$ rake db:migrate
```
run the rails server:
```
$ rails s
```
go to localhost:3000 in your web browser to see the app up and running.
